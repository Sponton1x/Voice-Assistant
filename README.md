## Installation:


     Linux/macOS
    git clone https://github.com/Sponton1x/AI.git
    cd Voice Assistant
    python3 main.py


     Windows
    git clone https://github.com/Sponton1x/AI.git
    cd Voice Assistant
    main.py
    
## Install PyAudio:

    Windows
    pip install pipwin
    pipwin install pyaudio

    Linux/macOS
    python3 pip install pipwin
    pipwin install pyaudio
